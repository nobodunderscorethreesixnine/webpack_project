Practicing Webpack Project

this code below is recommended or standardized code, u should do this way
"scripts": {
"build": "webpack --config webpack.prod.js",
"predev": "echo 'Starting server'",
"dev": "webpack serve --config webpack.dev.js",
"deploy": "git subtree push --prefix dist origin gh-pages"
}
but u can also do like this
"scripts": {
"build": "webpack --mode production",
"predev": "echo 'Starting server'",
"dev": "webpack serve",
"deploy": "git subtree push --prefix dist origin gh-pages"
}
this is i learn from this article -> https://www.upgrad.com/blog/introduction-to-package-json-scripts-in-node-js/
